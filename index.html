Lists

Lists behave like any other text for the most part but there are a few specific CSS properties for <ol> and <ul> list (no other styles applied to <dl> lists)

  • list-style-type
  • list-style-position
  • list-style-image
  • list-style

list-style-type

...allows you to set what type of bullet to use for bullet points, or numbers, or letters.

ol {
list-style-type: upper-roman;
} ul {
list-style-type: square;
}

list-style-position

...sets whether the bullets appear inside the list items or outside them (default) before the start of each item .

ol {
list-style-type: upper-roman;
list-style-position: inside;
}
  1. inside
  2. outside
