Lists

Lists behave like any other text for the most part but there are a few specific CSS properties for <ol> and <ul> list (no other styles applied to <dl> lists)

  • list-style-type
  • list-style-position
  • list-style-image
  • list-style

list-style-type

...allows you to set what type of bullet to use for bullet points, or numbers, or letters.

ol {
list-style-type: upper-roman;
}
ul {
list-style-type: square;
}

list-style-position

...sets whether the bullets appear inside the list items or outside them (default) before the start of each item .

ol {
list-style-type: upper-roman;
list-style-position: inside;
}
  1. inside
  2. outside

list-style-image

...allows you to use a custom image for your bullet..

ul {
list-style-image: url(star.svg);
}

This property is limited in terms of controlling position, size, etc and instead can use the background family of properties:

ul {
padding-left: 2rem;
list-style-type: none;
}

ul li {
padding-left: 2rem;
background-image: url(star.svg);
background-position: 0 0;
background-size: 1.6rem 1.6rem;
background-repeat: no-repeat;
}

list-style

... is the shorthand to set list-style-type, list-style-position and list-style-image.

ul {
list-style-type: square;
list-style-image: url(example.png);
list-style-position: inside;
}

could instead be...

ul {
list-style: square url(example.png) inside;
}

The values can be listed in any order and you can use one, two or all three (default for values not specified are disc, none and outside).

::marker

This pseudo-element is used to style the markers of list items and works on all display:list-item like <li> and <summary> (can also set elements like headers <h3> to display:list-item)

Links

Link states are:

  • link (unvisited)
  • visited
  • hover
  • focus (keyboard select)
  • active

Tip: to test links use # instead of the url to link back to same page.

Default link stying:

  • Links are underlined
  • Unvisited links are blue
  • Visited links are purple
  • Hovering over a link will make the mouse pointer change to a little hand icon
  • Focused links will have an outline around them
  • Active links are red

The order the links are styled in is IMPORTANT

a {…}
a:link {…}
a:visited {…}
a:focus {…}
a:hover {…}
a:active {…}

A good practice is to include icons in links to show what type of link it is (opens external site for example)

a[href*="http"] {
background: url('https://mdn.mozillademos.org/files/12982/external-link-52.png') no-repeat 100% 0;
background-size: 16px 16px; padding-right: 19px;
}
